- Install Java and JDK
- In IntelliJ,  *File > Project Structure*
- Click *Project* under *Project Settings*
- Click *SDK* dropdown
- Hover over *+ Add SDK*
- Click *JDK*
- You'll probably find the jdk in /Library/Java/JavaVirtualMachine/{version}/Contents/Home
- Click Apply
